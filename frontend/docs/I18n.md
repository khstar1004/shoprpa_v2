# 항목목록항목항목 (i18n) 항목문서

## 개요

본항목목록사용 `@lobehub/i18n-cli` 도구및 `i18next` 라이브러리항목항목항목공가능, 지원중국어(zh-CN)및영어(en-US)항목항목.

## 디렉터리결과항목

```
locales/
├── package.json       # 항목패키지매칭항목
├── zh-CN.json        # 중국어항목패키지(항목항목패키지)
└── en-US.json        # 영어항목패키지(항목항목항목성공)
```

## 매칭항목파일

### 1. `.i18nrc.cjs` - i18n 도구매칭항목

```javascript
const { env } = require('node:process')

const { defineConfig } = require('@lobehub/i18n-cli')

const modelName = env.OPENAI_MODEL_NAME || 'azure/gpt-4o-mini'

module.exports = defineConfig({
  modelName, // AI 항목항목유형
  entry: 'locales/zh-CN.json', // 항목항목파일
  entryLocale: 'zh-CN', // 항목항목
  output: 'locales', // 출력디렉터리
  outputLocales: ['en-US'], // 목록항목항목
  saveImmediately: true, // 매개항목항목성공후항목저장
  concurrency: 5, // 항목발송항목데이터량
  temperature: 0.3, // AI 항목항목정도매개변수
})
```

## 항목목록항목방법항목

### 1. 항목패키지로드 (`packages/web-app/src/config/i18n.ts`)

```typescript
import enUS from '@rpa/locales/en-US.json'
import zhCN from '@rpa/locales/zh-CN.json'

const locales = {
  'en-US': {
    lng: 'en-US',
    lang: 'English',
    value: enUS,
  },
  'zh-CN': {
    lng: 'zh-CN',
    lang: '한국어',
    value: zhCN,
  },
}

export default locales
```

### 2. i18next 항목항목 (`packages/web-app/src/plugins/i18next.ts`)

```typescript
import i18next from 'i18next'

// 항목감지: 항목사용중국어, 항목항목사용영어
const language = navigator.language === 'zh-CN' ? 'zh-CN' : 'en-US'

i18next.init({
  debug: false,
  lng: language,
  resources: {
    [language]: {
      translation: locales[language].value,
    },
  },
  interpolation: {
    escapeValue: false,
  },
})

// 항목항목데이터
export function translate(str: string | languageType, variables?: Record<string, string>): string {
  // 지원문자열항목값, 예: t('tableTotal', { count: 10 })
}

// 항목항목
export async function changeLanguage(lang: string) {
  i18next.addResourceBundle(lang, 'translation', locales[lang].value)
  await i18next.changeLanguage(lang)
}
```

### 3. Vue 컴포넌트중의사용

항목목록중항목필요있음항목사용방법방식: 

1. **사용 `useTranslation` hook**(권장사용법)
2. **직선연결가져오기항목지정항목데이터**(예항목항목공가능)

#### 항목중사용: 

```vue

  <!-- 항목항목 -->
  <div>{{ $t('projectName') }}</div>

  <!-- 항목매개변수의항목 -->

  <!-- 에서속성중사용 -->
  <a-input :placeholder="$t('enterProjectNameKeyword')" />

  <!-- 가져오기현재항목 -->
  <div>{{ $i18next.language }}</div>
</template>
```

#### Script 중사용: 

```vue
<script setup lang="ts">
import { useTranslation } from 'i18next-vue'

// 사용 useTranslation hook 가져오기항목항목데이터
const { t, i18next } = useTranslation()

// 에서항목중사용항목

  { required: true, message: t('projectNameCannotBeEmpty') },
  { max: 32, message: t('cannotExceed32Chars') }

function showMessage() {
  message.success(t('createSuccess'))
}

// 가져오기현재항목
console.log(i18next.language)
</script>
```

#### 항목항목컴포넌트: 


## 항목항목

### 1. 추가새의항목텍스트

1. **에서중국어항목패키지중추가새항목목록**

   ```bash
   # 항목 locales/zh-CN.json
   {
     "newKey": "새의중국어텍스트",
     "withVariable": "패키지항목변수의텍스트: {{value}}"
   }
   ```

2. **실행항목명령항목성공영어버전**

   ```bash
   pnpm run i18n
   ```

   > 해당명령항목호출 `env-cmd lobe-i18n locale`, 사용 AI 항목항목추가의텍스트

3. **에서항목코드중사용**

   ```vue
   <!-- 항목중 -->
   <div>
    {{ $t('newKey') }}
    </div>

   <div>
    {{ $t('withVariable', { value: 'test' }) }}
    </div>

   <!-- Script 중 -->
   <script setup>
   import { useTranslation } from 'i18next-vue'

   const { t } = useTranslation()
   console.log(t('newKey'))
   </script>
   ```

### 2. 수정항목있음항목

1. **수정중국어항목패키지** (`locales/zh-CN.json`)
2. **실행항목명령항목업데이트영어버전**
   ```bash
   pnpm run i18n
   ```
